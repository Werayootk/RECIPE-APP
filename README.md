### Setup

1. Clone this repo
   ```sh
   git clone https://github.com/Your_username/RECIPE-APP.git
   ```
2. Install NPM packages in folder
   ```sh
   npm install
   ```
3. Get API key from 
```
https://developer.edamam.com/edamam-recipe-api
```
4. insert your key in folder redux file `api.js`
```
const YOUR_APP_KEY = "YOUR_APP_KEY";
const YOUR_APP_ID = "YOUR_APP_ID";
```
